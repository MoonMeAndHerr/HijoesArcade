# 🕹️ Hijoe's Arcade

Welcome to **Hijoe's Arcade**! This is a custom-built, interactive Python application designed specifically for **Team Hijoe's** exhibition booth at the **Data Challenge 5.0** live pitching event at UKM. 

While our main project (*The Talent-Economy Nexus*) handles the heavy macroeconomic forecasting, this arcade interface serves as our complementary "wow" factor—designed to attract judges and audience members, offering them a gamified, hands-on experience at our booth.

---

## ✨ Features

* 👾 **Retro-Modern Aesthetic:** A sleek, dark neon interface running at a crisp 1920x1080 resolution.
* 🖱️ **Dynamic UI/UX:** Interactive buttons with smooth hover animations and color transitions.
* 🎵 **Integrated Audio:** Built-in background music loop and interactive sound effects for button clicks.
* 🔇 **Audio Control:** A real-time mute/unmute toggle system so booth operators can control volume levels based on the exhibition hall's noise level.
* 🚀 **Modular Game Launcher:** Designed as a main menu hub to easily hook into various Python-based mini-games or modules.

---

## 🛠️ Technology Stack

* **Language:** Python 3
* **Graphics & Audio Engine:** `Pygame`
